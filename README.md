# Simulador de Compra de Vehículos

- Aplicación web desarrollada en JavaScript Vanilla que simula la selección y compra de vehículos en un concesionario.

# Descripción del Proyecto

- Visualizar una lista de vehículos disponibles.

- Seleccionar un vehículo.

- Guardar la selección en localStorage.

- Calcular el precio final según financiación en cuotas.

# Tecnologías Utilizadas

- HTML5

- JavaScript (ES6)

- DOM Manipulation

- LocalStorage
